Последовательность представлена массивом структур:

```c
struct Date {
```

**Пример входных данных:**
```
5

1977 02 01
1994 03 01
2004 02 29
1977 08 01
```

**Пример выходных данных:**
```
1977 02 01
1977 08 01
1994 03 01
2004 02 29
2005 01 12
```

**Файл:** [prefixes.c](prefixes.c)

Составьте программу `prefixes.c`, выполняющую поиск всех периодических префиксов строки S. Префикс является периодическим, если его можно представить в виде d, повторённого k раз. Программа строит префиксную функцию для строки S.

Строка S передаётся через аргументы командной строки. Для каждого найденного префикса выводится длина префикса n и количество повторений k подстроки d.

**Формат входных данных:**
- Первая строка: размер последовательности n (0 < n ≤ 1 000 000)

- `UPD i v` — присвоить значение v элементу с индексом i
- `END` — последняя операция

**Формат результата:** для каждой операции `MAX` вывести значение максимального элемента.
